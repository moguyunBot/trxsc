# TRX靓号地址生成器WEB版

客服QQ:1187242467
支持各种定制开发

支持定义多个指定后最在线生成,后台会记录生成成功的地址和私钥
支持定制开发,全开源

# 安装说明
* 1.先安装([蘑菇云框架](https://github.com/moguyunBot/moguyun.pro))
* 2.把此插件放到蘑菇云框架的public/addons目录下
* 3.进入后台即可在应用中心看到此插件,点击安装
* 4.进入插件设置好域名,即可通过该域名访问
